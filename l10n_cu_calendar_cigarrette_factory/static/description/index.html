Planifique orientado a actividades y que el sistema genere la información según los modelos establecidos

Plan Anual de Actividades

El sistema permite realizar la planificación anual de actividades a realizar en la empresa por los grupos o de forma individual por trabajadores. Le permite utilizar tareas recurrentes para facilitar la planificación. Permite imprimir el plan anual de actividades (Modelo 1) según modelo establecido en la legislación vigente.

Plan de trabajo grupo

El sistema construye automáticamente el plan de trabajo de los grupos según las tareas asignadas a los participantes en las tareas. De esta forma es más eficiente en cuanto a la gestión del plan de trabajo del grupo. Permite imprimir el plan de trabajo del grupo (Modelo 2) según modelo establecido en la legislación vigente.

Plan de trabajo individual

Resumen de cumplimiento del plan de trabajo individual

El sistema permite imprimir un resumen del cumplimiento de plan de trabajo individual con la cantidad de tareas cumplidas, incumplidas, modificadas y tareas extraplan. En cada caso se muestra una tabla con los detalles, de las tareas incumplidas con las causas, las modificaciones realizadas y un listado de las tareas extraplan.

Compatible con estándar ICalendar

A través de un asistente el sistema permite exportar un fichero con todas las tareas en un periodo. De esta forma usted puede copiar el fichero en un dispositivo móvil y llevar a todas parte su plan de trabajo individual permitiendo la rápida consulta y las alertas más oportunos. Solo debe asegurarse que el dispositivo móvil cuente con una aplicación compatible con este estándar.
